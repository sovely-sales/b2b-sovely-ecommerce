## Instructions to use the extension

### 1. Open Chrome browser, go to the extensions page, and enable developer options.
### 2. Click Load Unpacked, and a popup will appear.
### 3. Select the folder containing the extensions ('extension' folder if accessing from repo)
### 4. Go to Wukusy, sign in, go to Orders page, click on All Orders, and a button will appear on the top right of the page.
### 5. Click the Export All Orders button on the Wukusy page, and a csv will get downloaded.
### 6. Click on the Augment Orders Csv button, a popup will appear.
### 7. Enter the location of the downloaded csv file from the last step. The augmented csv will get downloaded.
